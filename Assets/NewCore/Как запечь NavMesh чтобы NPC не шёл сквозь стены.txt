Как запечь NavMesh — чтобы клиент обходил стены и не летел сквозь них

1. Открой окно: Window → AI → Navigation (или Window → Navigation в старых версиях).

2. Вкладка "Bake":
   - Agent Radius — примерно 0.3–0.5 (чтобы NPC не застревал).
   - Agent Height — рост агента (например 2).
   - Max Slope — до 45, если есть лестницы/склоны.
   Нажми кнопку "Bake" внизу.

3. Кто участвует в запекании:
   - Пол/земля: выбери в Hierarchy, в Inspector включи галочку "Static" (или в окне Navigation → вкладка Object → включи "Navigation Static"). Пол должен быть запечён как проходимая зона.
   - Стены, столы, стулья: тоже отметь как "Navigation Static" (или "Static"). Тогда при Bake они будут "вырезать" проходимую зону — синяя сетка будет только там, где можно ходить, и NPC будет обходить препятствия.

4. После Bake на полу (и проходимых местах) появится синяя сетка — NavMesh. NPC с NavMeshAgent должен стоять на этой сетке (у двери на полу). Точка стойки (AdminCounterSpot) тоже должна быть на синей сетке.

5. У NPC обязательно компонент Nav Mesh Agent (Add Component → Nav Mesh Agent). Скорость можно выставить в Client NPC (Move Speed) или в самом Nav Mesh Agent (Speed).

Готово: клиент будет идти к стойке в обход стен и мебели.
